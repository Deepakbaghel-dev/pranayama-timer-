\# Pranayama Timer



A Python-based breathing timer inspired by traditional pranayama practice.



\## Features



-Runtime-configurable breathing durations

-Repeating breathing cycles

-Audio cue at the end of each phase

-Terminal-based execution



\## Tech Stack 



-Python

-time module

-winsound (windows only) 



\## Usage



Run the script and follow the prompt to set breathing cycles and you can manually change the duration of each phase directly according to your body and phase you are in
